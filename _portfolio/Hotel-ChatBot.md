## Funcionalidades principales
- Responde consultas de forma automática.
- Registra todas las conversaciones en Google Sheets mediante Google Apps Script.
- Diseño responsivo adaptable a móviles y tablets.
- Botones de acción rápida para facilitar la interacción del huésped.

## Tecnologías usadas
- Frontend: HTML / CSS / JavaScript
- Backend / Integraciones: Google Apps Script
